Projekte

Geo Finder

Dieses Projekt nutzt Geodaten-APIs, um über eingegebene Ortsnamen die zugehörigen Koordinaten zu ermitteln und eine Route zwischen zwei Orten zu berechnen – inklusive Entfernung und geschätzter Fahrtdauer. Die berechnete Route wird dabei visuell auf einer interaktiven Karte dargestellt. Das Backend basiert auf Flask und kommuniziert mit den OpenStreetMap- und OpenRouteService-Schnittstellen. Das Frontend wurde mit React, TypeScript und Vite umgesetzt und auf GitHub Pages gehostet.

Recipe Book

Dieses Projekt ist eine interaktive Rezeptverwaltungs-App, die es Nutzer:innen ermöglicht, eigene Kochrezepte zu erstellen, zu durchsuchen, zu bewerten und zu speichern. Zusätzlich können per Spoonacular-API zufällige Rezepte aus dem Internet geladen und übernommen werden. Alle Inhalte werden übersichtlich dargestellt, inklusive Zutatenliste, Zubereitungszeit, Bild und Anleitung. Die Anwendung basiert auf Blazor Server (ASP.NET Core), nutzt Bootstrap für das Styling und ist über Render.com öffentlich zugänglich.

Pathfinding Visualizer

Dieses Projekt visualisiert vier verschiedene Pathfinding-Algorithmen. Per Knopfdruck wird ein zufälliges Labyrinth erzeugt, das garantiert einen gültigen Pfad vom Start- zum Endpunkt enthält. Durch Mausklick lassen sich gezielt Mauern entfernen, um das Verhalten der Algorithmen auch in Labyrinthen mit Schleifen zu testen. Die Anwendung wurde mit React und TypeScript umgesetzt.

Food Detect

Dieses Projekt nutzt eine Bilderkennungs-API, um über eine eingegebene Bild-URL das dargestellte Essen zu identifizieren – inklusive Angabe, wie wahrscheinlich die Erkennung korrekt ist. Die Anwendung enthält außerdem ein Authentifizierungssystem mit Registrierung und Login, verwaltet über ein Node.js-Backend. Das Frontend wurde mit React und JavaScript umgesetzt. Das Eingabefeld benötigt ein gültiges Bildformat, diese Beispielurl eines Apfels funktioniert immer: https://png.pngtree.com/png-vector/20231017/ourmid/pngtree-fresh-apple-fruit-red-png-image_10203073.png
